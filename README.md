# RentHome

Учебное Java web-приложение для курсового проекта по теме "Сервис аренды жилья".

## Возможности

- регистрация и вход пользователей;
- роли: арендатор, владелец, администратор;
- создание, просмотр, поиск и редактирование объявлений;
- избранное для арендаторов;
- заявки на аренду и смена статуса владельцем;
- отзывы к объявлениям;
- простая административная модерация объявлений.

## Запуск

```powershell
.\scripts\run.ps1
```

После запуска откройте:

```text
http://localhost:8090
```

Если порт занят, можно указать другой:

```powershell
.\scripts\run.ps1 8090
```

## Демо-аккаунты

| Роль | Email | Пароль |
| --- | --- | --- |
| Администратор | admin@renthome.local | admin |
| Владелец | owner@renthome.local | owner |
| Арендатор | tenant@renthome.local | tenant |

## Структура

- `ru.renthome.controller` - обработка HTTP-запросов;
- `ru.renthome.service` - бизнес-логика;
- `ru.renthome.repository` - хранение и поиск данных;
- `ru.renthome.entity` - доменные сущности;
- `ru.renthome.util` - сессии, HTML, пароли и обработка форм.
